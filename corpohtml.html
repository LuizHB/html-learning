Estrutura de um código HTML

A estrutura do HTML é definida por:

O corpo do código em HTML se divide em 2 áreas principais.

  • Head: Local de configuração com informações que não aparecem diretamente para o usuário.
  • Body: Local de configurações que aparecem direto para o usuário.

Atributos

São informações adicionais em relação aos elementos utilizados.
Exemplo: Temos a tag img utilizada para incluir uma imagem numa página HTML. O atributo src especifica o caminho de onde a imagem tem origem.
<img src="imagem.jpg">
