Comandos

Antes da estrutura do código podemos definir o tipo de documento utilizando a TAG <!DOCTYPE html>

No Head, na parte do título, utilizamos um comando para definição da codificação do texto utilizado no HTML. Assim após "title" utilizaremos "<meta charset="utf-8">".

Para os títulos temos os seguintes tamanhos definidos para o HTML. De <h1> a <h6>.

  • <h1> Título 1 </h1>

  • <h2> Título 2 </h2>

  • <h3> Título 3 </h3>

  • <h4> Título 4 </h4>

  • <h5> Título 5 </h5>
  • <h6> Título 6 </h6>

Formatação de texto

  • <b> Texto em negrito </b>
  • <strong> Texto importante </strong>
  • <i> Texto em itálico </i>
  • <em> Texto destacado </em>
  • <mark> Texto marcado </mark>
  • <small> Texto diminuído </small>
  • <del> Texto deletado </del>
  • <ins> Texto inserido </ins>
  • <sub> Texto subscrito </sub>
  • <sup> Texto sobrescrito </sup>

Mais utilizados

<p> Para criar um parágrafo </p>
<br> Pular linha
<hr> Para criar uma barra divisória
<a> Âncora para outro link </a> Sempre com o atributo href="" que designa o endereço do hyperlink. Exemplo: <a> href="conta.html" </a>
<img> Para adicionar imagens. Possui vários atributos. O principal é o src em que se define o endereço da imagem, height para altura, width para largura, entre outros.

Tabelas

<table> Para criar uma tabela </table>
Diversos atributos da tabela são: border (tipo de borda da tabela), height (altura máxima da tabela em relação à página), width (largura máxima em relação à pagina), align (alinhamento da tabela).
Além disso temos: <tr> Item tabelado </tr> em que se divide em duas categorias de items:
<th> Título da coluna</th> e <td> objeto da coluna </td> (sendo que precisam estar em <tr> diferentes para cada linha).

Listas

  • <ul>Lista não ordenada </ul>
  • <ol>Lista ordenada </ol>
  • <dl>Lista de descrição </dl>
Utilizando o atributo type="" para definir o tipo de estilo para a lista (números, letras, círculos, quadrados, entre outros).
Os itens das listas ordenada e não ordenada são determinadas por <li> item </li>.
A lista de descrição utiliza <dt> termo </dt> determina o termo e <dd> informação </dd> descreve cada termo.

Entidades

São comandos que reescrevem os códigos na forma de texto. Exemplo: "& lt;" serve para < e "& gt;" serve para >. Ambos os termos dentros das aspas sem espaço.
